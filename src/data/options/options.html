In short
Google™ Translator is a powerful tool to translate selected words (or sentences) to a wide range of supported languages.
Features
- Easy One-Click Translation via Google Translator.
- Supports All Available Languages.
- Auto-Detect Input Language.
- Auto-Correction of Input Language.
- Pop-up Bubble Translator.
- Selected-Text Translator by holding Alt key.
- Context-Menu Translator.
Some useful notes
[1] Before using the extension, please make sure to adjust your settings.
[2] All changes in the settings page are auto saved.
[3] To activate the extension after installation, please reload all your existing pages.
[4] To help a humble monetization campaign of the application please visit Google™ Translator and use the Donation button.
